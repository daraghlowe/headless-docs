# wpe apps

The `wpe apps` command group provides functionality for managing your WP Engine Headless Applications.

## Commands

| Command              | Description                                                                   |
| :------------------- | :---------------------------------------------------------------------------- |
| [create](./create)   | Create a new WP Engine Headless Application                                   |
| [delete](./delete)   | Remove an WP Engine Headless Application                                      |
| [get](./get)         | Get information about an existing WP Engine Headless Application              |
| [init](./init)       | Generate a `wpe.json` file for deploying a new WP Engine Headless Application |
| [list](./list)       | List all existing WP Engine Headless Applications                             |
| [rebuild](./rebuild) | Rebuild an existing WP Engine Headless Applications                           |
| [update](./update)   | Update an existing WP Engine Headless Application                             |
