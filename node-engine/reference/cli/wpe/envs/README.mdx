# wpe envs

The `wpe envs` command group allows you to manage environments in an application using the WP Headless CLI.

## Commands

| Command              | Description                                                             |
| :------------------- | :---------------------------------------------------------------------- |
| [delete](./delete)   | Remove a deployed environment from an application                       |
| [purge](./purge)     | Manually purge the domain cache for a specified application environment |
| [rebuild](./rebuild) | Manually invoke a rebuild of a specified application environment        |
