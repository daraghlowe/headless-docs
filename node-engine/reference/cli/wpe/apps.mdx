# wpe apps

The `wpe apps` command group provides functionality for managing your WP Engine Headless Applications.

## Commands

- [**wpe apps create**](./apps/create): Create a new WP Engine Headless Application
- [**wpe apps delete**](./apps/delete): Remove an WP Engine Headless Application
- [**wpe apps get**](./apps/get): Get information about an existing WP Engine Headless Application
- [**wpe apps init**](./apps/init): Generate a `wpe.json` file for deploying a new WP Engine Headless Application
- [**wpe apps list**](./apps/list): List all existing WP Engine Headless Applications
- [**wpe apps rebuild**](./apps/rebuild): Rebuild an existing WP Engine Headless Applications
- [**wpe apps update**](./apps/update): Update an existing WP Engine Headless Application
