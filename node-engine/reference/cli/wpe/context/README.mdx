# wpe context

The `wpe context` command group allows you to get and set account context for the WP Headless CLI.

## Commands

| Command      | Description                                            |
| :----------- | :----------------------------------------------------- |
| [get](./get) | Get the account context for the WP Engine Headless CLI |
| [set](./set) | Set the account context for the WP Engine Headless CLI |
