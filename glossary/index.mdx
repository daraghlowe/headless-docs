This glossary provides definitions for terms related to headless WordPress.

## A

### Atlas

The complete headless WordPress platform. Atlas includes dynamic Node.js and headless WordPress in one package with your choice of modern front-end framework.

### Atlas Content Modeler

An open-source project from WP Engine that simplifies the process of creating content models for your headless WordPress site. With a single tool, you can create a new content model, define the custom fields and taxonomies for it, develop relationships between teh model and other posts or users, adn have all of that data automatically available to you via the WordPress REST API or WP GraphQL.

## B

### Backend

The server-side development, focused on how the site works.

## C

### Content Management System (CMS)

A web application that is used to manage content. WordPress is the most popular CMS, powering 37% of the web.

### Component

A reusable custom element in web development.

### Config

A file in your site's directory that defines the options, settings, and preferences for your project.

### CSS

A style sheet language used to change the presentation and design of a webpage.

## D

### Database

An organized collection of data.

### Decoupling

The separation of the backend from the frontend.

## E

## F

### Faust.js

An open-source JavaScript framework for headless WordPress. Faust.js consists of a WordPress plugin, a set of npm packages, and guides to get you started building headless WordPress sites.

### Frontend

The development of the presentation and appearance of a website.

## G

### GQTY

A proxy-based GraphQL client built for rapid iteration. GQTY performs an invisible or skeleton render to identify what data is needed.

### GraphQL

An open-source data query and manipulation language.

## H

### HTML

The standard markup language for websites.

### Headless CMS

A content management system where the backend content repository (the "body") is decoupled from the presentation layer (the "head").

### Headless WordPress

A website that uses WordPress for the backend content repository with some other custom frontend framework.

An open-source JavaScript development framework built on top of Node.js enabling React-based web application functionalities.

An open-source JavaScript development framework built on top of Node.js enabling Vue-based web application functionalities.

## S

### Schema

The organization or structure of a database.

### Svelte

An open-source front-end compiler. Svelte shifts the work traditionally done in the browser into a compile step that happens when you build your app.

### SvelteKit

A framework for building web applications.

An open-source content management system. WordPress powers more than 40% of the top 1 million websites by traffic.
