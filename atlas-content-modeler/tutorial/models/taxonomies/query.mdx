## Query WPGraphQL for a Taxonomy

Learn how to query taxonomies using WPGraphQL.

---

### Dependencies

#### WPGraphQL

Please download and install WPGraphQL following these instructions: [WPGraphQL Installation - Quick Start](https://www.wpgraphql.com/docs/quick-start/).

#### WPGraphQL Tax Query

Please download, install, and activate **WPGraphQL Tax Query** using the [Wordpress Plugin Installation Method](https://wordpress.org/support/article/managing-plugins/#automatic-plugin-installation). Documentation: [WPGraphQL Tax Query Github](https://github.com/wp-graphql/wp-graphql-tax-query).

Once you have successfully installed and activated these plugins, you may continue with the steps below for Querying Taxonomies.

---

### Steps to Query a Taxonomy using WPGraphQL

You can find detailed steps in the basic usage of WPGraphQL here: [WPGraphQL Quick Start]()

The steps below are more specific on how to query taxonomies once you are in WPGraphQL.

---

1. Click on Content Modeler in the WordPress side menu

   ![Atlas Content Modeler sync](../../../images/acm-sidebar.png)

1. If it's not on Taxonomies, click Taxonomies under Content Modeler in the WordPress side menu

   ![Atlas Content Modeler sync](../../../images/taxonomy-menu.png)

1. You will then see the form to Add a Taxonomy and the current Taxonomies table

   ![Atlas Content Modeler sync](../../../images/taxonomies.png)

1. Fill in the fields to fit your needs

1. Taxonomy Id is auto generated but you can change it to suite your needs

1. Click Create to save the Taxonomy, it will be added to the table
